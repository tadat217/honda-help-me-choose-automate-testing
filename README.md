# HONDA HELP ME CHOOSE MODULE AUTOMATE TEST

## Install

### 1. Create and run virtule enviroment

```
$ python -m venv .venv
$ source .venv/bin/activate # MacOS / Linux
$ .venv/Scripts/activate # Windows
```

### 2. Import package

```
$ pip install -r requirements.txt
$ playwright install
```

### 3. Run test

```
$ pytest
```
